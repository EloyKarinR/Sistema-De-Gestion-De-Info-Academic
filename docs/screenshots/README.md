# Capturas de pantalla para el README

Coloca aquí las capturas con estos nombres exactos — el `README.md` de la raíz ya las referencia, así que en cuanto las agregues van a aparecer solas, sin tocar nada más.

| Archivo | Qué capturar |
|---|---|
| `dashboard.png` | El Dashboard principal, logueado como Administrador. |
| `matricula.png` | El flujo de "Nueva Matrícula" — idealmente el paso 3 (elegir aula) o el recibo final. |
| `estudiante-ficha.png` | La ficha completa de un estudiante (con foto, datos, historial). |
| `portal-acudiente.png` | El Portal del acudiente, mostrando notas y/o horario de un hijo. |
| `horario.png` | La tabla de horario semanal de un aula (desde Académico o la ficha del estudiante). |
| `boletin.png` | El PDF del boletín de notas abierto en la vista previa. |
| `docentes.png` | El listado de Docentes. |
| `academico.png` | La pantalla de Académico mostrando las aulas agrupadas por turno. |

**Recomendaciones:**
- Formato PNG o JPG, cualquier resolución razonable (el README las muestra a 50% de ancho en pantallas grandes).
- Si es posible, usa el modo oscuro de la app para las capturas — es el que se ve mejor en GitHub.
- Evita que se vean datos sensibles reales (usa los datos de práctica que ya vienen sembrados).
